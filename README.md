# JPContext-Client

## About
JPContext is a language learning application where Japanese language learners can further their education by listening to real examples of vocabulary terms being used in context by native speakers. 

## Getting Started

### Prerequisites
- Node.js (v18 or higher)
- npm or yarn package manager

### Installation

1. Clone the repository
2. Install dependencies
    ```bash
    npm install
    ```
3. Start the development server
    ```bash
    npm run dev
    ```

4. Open [http://localhost:3000](http://localhost:3000) in your browser


### Backend Setup
The client application requires a backend API running at `http://localhost:5000`. See the backend repository for setup instructions:
[https://github.com/DuncanMcPherson903/JPContext-API]
